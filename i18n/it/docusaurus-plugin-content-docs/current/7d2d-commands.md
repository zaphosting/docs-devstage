## La Console
Per eseguire comandi sul tuo server, devi essere un admin. Se non sai come diventarlo, dai un’occhiata alla nostra guida.
Tutti i comandi devono essere eseguiti tramite la console degli sviluppatori. Puoi aprirla premendo `F1`

La console si presenta così:

![](https://screensaver01.zap-hosting.com/index.php/s/Lyi4qrT5MjzJLkL/preview)

## I Comandi
I comandi più importanti:

| Comando    | Descrizione                                                 |
| ----------------- | ------------------------------------------------------------ |
| giveself [id oggetto] [livello qualità 1-6] | Aggiunge un oggetto a te stesso, con il livello di qualità scelto |
| debugmenu on/off | Attiva il Debug Menu, si apre premendo G |
| giveselfxp [quantità] | Aggiunge punti XP al tuo personaggio |
| creativemenu on/off | Attiva il Creative Menu, si apre premendo U |
| spawnsupplycrate | Fa spawnare una supply crate nella tua posizione |
| admin [add] [remove] [list] | Comando per aggiungere/rimuovere admin |
| kick [nome giocatore] [motivo] | Espelle un giocatore dal server |
| tele [Giocatore A] [Giocatore B] | Teletrasporta il Giocatore A dal Giocatore B |
| weather | Cambia le condizioni meteo |
| help | Mostra tutti i comandi disponibili |

[altri comandi](https://commands.gg/7dtd)
