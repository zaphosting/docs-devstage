## Créer un compte
Créer un compte ZAP-Hosting est la première étape pour accéder et gérer tous les services et fonctionnalités. Le processus d'inscription est rapide et ne nécessite que quelques informations. Test

Pour commencer l'inscription, rendez-vous sur le site de ZAP-Hosting et allez sur la page d'accueil. Vous y trouverez la zone de connexion avec un bouton **S’inscrire !**. Cliquez sur **S’inscrire !** pour ouvrir la fenêtre popup d'inscription.

![img](https://screensaver01.zap-hosting.com/index.php/s/5b6Wzc27CcHW3Zx/download)


Dans la fenêtre popup, vous devez renseigner les informations suivantes :

- Nom d’utilisateur
- Adresse e-mail
- Mot de passe

Après avoir saisi vos informations, confirmez le captcha de sécurité. Cette étape est nécessaire pour protéger contre les inscriptions automatisées. Une fois tous les champs remplis et le captcha validé, envoyez le formulaire d'inscription pour créer votre compte.

![img](https://screensaver01.zap-hosting.com/index.php/s/k6P3wSKWWRRiYax/preview)

## Activation du compte

Après avoir soumis le formulaire d'inscription, un e-mail d'activation est envoyé à l'adresse e-mail fournie. Cet e-mail contient un code d'activation qui doit être confirmé pour finaliser la création du compte.

La réception de l’e-mail d’activation peut prendre quelques minutes. Si le message n’apparaît pas dans votre boîte de réception, vérifiez également votre dossier spam ou courrier indésirable.

:::warning Fonctionnalités limitées sans activation
Vous pouvez vous connecter immédiatement après l'inscription. Cependant, pour accéder à toutes les fonctionnalités et services, l’e-mail d’activation doit être confirmé.
:::
